# 球员数据页面修复计划

## 问题分析

当前文班亚马专区存在以下问题：
1. 生涯数据显示不正确，新秀赛季得分单独显示没有必要
2. 需要简化数据展示，只显示本赛季常规赛和季后赛平均数据
3. 各球队专区页面风格不统一
4. 部分页面文字与背景对比度不够

## 修复计划

### 1. 修改文班亚马API (api/spurs.py)
- 简化数据结构，移除不必要的字段
- 只返回本赛季(2025-26)常规赛和季后赛平均数据
- 数据结构：
  ```json
  {
    "success": true,
    "data": {
      "player_info": {...},
      "regular_season": {
        "avg_points": 22.8,
        "avg_rebounds": 9.2,
        "avg_assists": 2.6,
        "avg_blocks": 3.2,
        "avg_steals": 1.0,
        "games_played": 52
      },
      "playoffs": {
        "avg_points": 25.5,
        "avg_rebounds": 10.1,
        "avg_assists": 3.0,
        "avg_blocks": 3.8,
        "avg_steals": 1.2,
        "games_played": 12
      },
      "career_highlights": [...]
    }
  }
  ```

### 2. 修改马刺专区前端 (static/spurs.html)
- 移除新秀赛季得分对比卡片
- 移除联盟对比卡片
- 简化文班亚马专区布局，只显示：
  - 球员基本信息
  - 本赛季常规赛平均数据
  - 本赛季季后赛平均数据
  - 职业生涯亮点

### 3. 统一球队专区页面风格

| 球队 | 主背景色 | 文字颜色 | 强调色 |
|------|----------|----------|--------|
| 马刺 | 纯黑色 (#000000) | 银色 (#C4CED4) | 白色 (#FFFFFF) |
| 湖人 | 紫色 (#552583) | 金色 (#FDB927) | 白色 (#FFFFFF) |
| 火箭 | 红色 (#CE1141) | 白色 (#FFFFFF) | 银色 (#C0C0C0) |

### 4. 文件修改清单

| 文件 | 修改内容 |
|------|----------|
| api/spurs.py | 简化文班亚马API数据结构 |
| static/spurs.html | 简化文班亚马专区布局 |
| static/spurs.html | 统一页面风格为黑色+银色 |
| static/lakers.html | 统一页面风格为紫色+金色 |
| static/rockets.html | 统一页面风格为红色+白色 |

### 5. 风险处理

- **API兼容性**：修改API可能影响其他页面，需要确保前端同步更新
- **样式冲突**：统一风格可能影响现有组件样式，需要仔细测试
- **数据完整性**：确保季后赛数据有合理的默认值
